## 基础
    *MyBatis是一款ORM型的持久层框架。
        ORM：ORM的全称是Object Relational Mapping，即对象关系映射。
        ORM框架的主要功能就是根据映射配置文件，完成数据在对象模型与关系模型之间的映射。
    *可以在：https://mvnrepository.com 查找需要导入的依赖
    *MyBatis的技术文档：https://mybatis.org/mybatis-3/zh/index.html
